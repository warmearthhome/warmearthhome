## 🎨 Phase 2: 视觉体系测试

### 颜色系统检查

#### 主色（Warm Bronze `#D4A574`）
- [ ] **主按钮颜色**：应为暖金色 `#D4A574`
  - 首页 CTA 按钮
  - "Add to Cart" 按钮
  - "Shop Now" 按钮
- [ ] **产品价格颜色**：应为 `#D4A574`
- [ ] **按钮悬停效果**：颜色应加深（`#C49564`）

#### 背景色
- [ ] **页面背景**：应为白色 `#FFFFFF`（简洁干净）
- [ ] **卡片背景**：应为象牙白 `#FFFFF8`

#### 文字颜色
- [ ] **标题文字**：应为深灰色 `#1a1a1a`
- [ ] **正文文字**：应为 `#1a1a1a` 或 `#666666`

### 按钮样式检查

#### 主按钮（Primary）
- [ ] 背景色：`#D4A574` (Warm Bronze)
- [ ] 文字颜色：白色 `#FFFFFF`
- [ ] 圆角：`8px`
- [ ] 悬停效果：颜色加深 + 阴影 + 轻微上移

#### 次按钮（Secondary）
- [ ] 背景：透明
- [ ] 边框：`2px solid #D4A574`
- [ ] 文字颜色：`#D4A574`
- [ ] 悬停效果：填充背景色

### 卡片样式检查
- [ ] **产品卡片**：
  - 背景：象牙白 `#FFFFF8`
  - 圆角：`8px`
  - 悬停：阴影加深 + 上移效果
- [ ] **系列卡片**：
  - 相同的圆角和悬停效果

### 字体检查
- [ ] **标题字体**：Playfair Display（衬线字体）
- [ ] **正文字体**：Inter（无衬线字体）
- [ ] **字体大小**：
  - H1: 48px（桌面）/ 32px（移动）
  - H2: 32px（桌面）/ 24px（移动）
  - Body: 18px（桌面）/ 16px（移动）

### 间距检查
- [ ] 元素间距统一（8px 网格系统）
- [ ] 页面布局有适当的留白
- [ ] 移动端间距适配良好
