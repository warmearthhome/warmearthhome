## 🚀 上传步骤

### 方法 1：通过 FTP/文件管理器上传（推荐）

#### 步骤 1：找到主题目录
你的主题目录通常是：
```
wp-content/themes/blocksy-child/
```
或者如果是主主题：
```
wp-content/themes/blocksy/
```

#### 步骤 2：上传模板文件
1. 使用 FTP 客户端（如 FileZilla）或主机文件管理器
2. 导航到主题目录
3. 将以下文件上传到主题根目录：
   - `templates/wordpress/page-support.php` → `page-support.php`
   - `templates/wordpress/page-homepage.php` → `page-homepage.php`（如果还没有）

#### 步骤 3：验证文件权限
确保文件权限设置为 `644`（可读可写）

---

### 方法 2：通过 WordPress 主题文件编辑器

⚠️ **注意**：主题文件编辑器通常只能编辑现有文件，不能创建新文件。如果主题目录中已经有 `page-homepage.php`，你可以参考它来创建 `page-support.php`。

#### 步骤 1：打开主题文件编辑器
1. 登录 WordPress 后台
2. 进入 **外观** → **主题文件编辑器**
3. 选择你的主题（blocksy-child 或 blocksy）

#### 步骤 2：创建新文件
1. 点击 **选择要编辑的文件** 下拉菜单
2. 如果看到 **创建新文件** 选项，点击它
3. 输入文件名：`page-support.php`
4. 复制 `templates/wordpress/page-support.php` 的全部内容
5. 粘贴到编辑器中
6. 点击 **更新文件**

---

## ✅ 上传后操作

### 1. 选择页面模板
1. 进入 WordPress 后台 → **页面** → **所有页面**
2. 找到 **Support Center** 页面，点击编辑
3. 在右侧 **页面属性** 面板中，找到 **模板** 下拉菜单
4. 选择 **Support Center** 模板
5. 点击 **更新** 保存

### 2. 验证页面显示
1. 访问：https://warmearthhome.com/support-center/
2. 检查页面是否正确显示：
   - Hero 区域
   - Quick Links 卡片
   - FAQ 手风琴
   - Shipping & Returns 信息
   - Installation Guides 标签页
   - Contact 表单
   - Newsletter CTA

### 3. 检查 CSS 和 JS
确保以下文件已上传到主题目录：
- `assets/css/warm-earth-home.css`
- `assets/js/warm-earth-home.js`

如果还没有，请上传到：
```
wp-content/themes/blocksy-child/assets/css/warm-earth-home.css
wp-content/themes/blocksy-child/assets/js/warm-earth-home.js
```

## 🔍 故障排除

### 问题 1：模板选项不显示
**原因**：文件未正确上传或位置不对
**解决**：
- 确认文件在主题根目录（不是子目录）
- 确认文件名正确：`page-support.php`
- 确认文件开头有 `Template Name: Support Center` 注释

### 问题 2：页面样式不正确
**原因**：CSS 文件未加载
**解决**：
- 检查 `functions-weh-enhancements.php` 是否已添加到主题的 `functions.php`
- 检查 CSS 文件路径是否正确
- 清除浏览器缓存和 WordPress 缓存

### 问题 3：JavaScript 功能不工作
**原因**：JS 文件未加载或 jQuery 未加载
**解决**：
- 检查 `functions-weh-enhancements.php` 是否正确加载 JS
- 检查浏览器控制台是否有错误
- 确认 jQuery 已加载
