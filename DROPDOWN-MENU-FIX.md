# ✅ 下拉菜单文字颜色修复完成

## 问题描述

下拉菜单中的文字颜色过浅（几乎白色），背景是浅米色/灰白色，导致对比度不足，文字难以阅读。

## 修复内容

已在 `assets/css/warm-earth-home.css` 中添加以下 CSS 规则：

### 1. 下拉菜单背景和边框
- 背景色：`var(--weh-soft-ivory)` (浅米白色)
- 边框：1px 浅灰色边框
- 圆角：8px
- 阴影：中等阴影效果

### 2. 下拉菜单项文字颜色
- 文字颜色：`var(--weh-urban-slate)` (#3E4A52) - 深灰色
- 字体粗细：500（中等）
- 内边距：适当间距
- 过渡效果：平滑过渡

### 3. 悬停状态
- 背景色：`var(--weh-warm-sand)` (浅棕色)
- 文字颜色：`var(--weh-deep-clay)` (深棕色)

### 4. 当前项高亮
- 背景色：`var(--weh-warm-sand)`
- 文字颜色：`var(--weh-deep-clay)`
- 字体粗细：600（粗体）

## 兼容性

修复针对以下选择器：
- Blocksy 主题的导航菜单
- 通用 WordPress 菜单下拉
- 各种主题变体
