# 产品导入执行指南

## 已创建的文件

1. **import-products.php** - 产品导入脚本
2. **woo-products-sample.csv** - 8 个示例产品数据

## 执行步骤

### 方法 1：通过浏览器执行（推荐）

1. **上传文件到 WordPress 根目录**
   - 将 `import-products.php` 上传到网站根目录（与 `wp-config.php` 同级）
   - 将 `woo-products-sample.csv` 也上传到根目录

2. **访问导入页面**
   - 确保已登录 WordPress 后台（管理员账号）
   - 访问：`https://warmearthhome.com/import-products.php?weh_import=1`
   - 脚本会自动执行导入

3. **查看结果**
   - 页面会显示导入进度和结果
   - 成功导入的产品会显示绿色提示
   - 如果产品已存在，会跳过并显示蓝色提示

4. **删除脚本（重要）**
   - 导入完成后，请删除 `import-products.php` 文件以确保安全

### 方法 2：通过 FTP 上传

如果无法通过浏览器上传，可以使用 FTP：

1. 使用 FTP 客户端连接到服务器
2. 上传 `import-products.php` 到 WordPress 根目录
3. 上传 `woo-products-sample.csv` 到 WordPress 根目录
4. 访问导入 URL 执行导入

## 注意事项

- 脚本会自动创建分类（如果不存在）
- 图片 URL 需要可访问，否则图片不会导入
- 如果产品 SKU 已存在，会跳过该产品
- 导入后检查产品分类和标签是否正确

## 验证导入

导入完成后，请检查：

1. **产品列表**：WordPress 后台 > 产品 > 所有产品
2. **分类页面**：前台访问各分类页面，确认产品显示
3. **标签页面**：确认产品标签正确关联
