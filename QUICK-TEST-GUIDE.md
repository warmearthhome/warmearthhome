# 快速测试指南

## 🚀 5 分钟快速测试

### 1. 清除缓存（1分钟）
- 打开 WordPress 后台 > LiteSpeed Cache > 清除全部缓存
- 或在浏览器按 `Ctrl + Shift + Delete` 清除缓存
- 使用无痕模式访问网站（推荐）

### 2. 检查导航菜单（1分钟）
访问 `https://warmearthhome.com`，检查顶部导航：

✅ **应该看到**：
- Shop
- My Account
- Checkout
- Cart

❌ **不应该看到**：
- 商店
- 我的帐户
- 结账
- 购物车

### 3. 检查主色（1分钟）
查看网站上的按钮和价格：

✅ **主色应该是**：暖金色 `#D4A574`

检查位置：
- 首页的 "Shop Now" 或 "Explore" 按钮
- 产品卡片的价格
- "Add to Cart" 按钮

### 4. 检查视觉一致性（2分钟）
快速浏览页面，检查：

✅ **字体**：
- 标题：Playfair Display（优雅的衬线字体）
- 正文：Inter（现代的无衬线字体）

✅ **风格**：
- 简洁、温暖
- 间距统一
- 圆角统一（8px）

---

## ⚠️ 如果发现问题

### 导航菜单仍有中文
- 刷新页面（Ctrl + F5）
- 清除所有缓存
- 检查是否使用了缓存插件

### 颜色没有更新
- **检查 CSS 文件是否已上传**：
  - WordPress 后台 > 外观 > 主题编辑器
  - 找到 `warm-earth-home.css` 文件
  - 检查主色是否为 `#D4A574`

- **如果 CSS 文件未更新**：
  - 需要上传更新后的 CSS 文件
  - 或手动更新文件内容

### 视觉效果不一致
- 清除浏览器缓存
- 检查是否有其他 CSS 文件覆盖样式
- 检查主题设置是否有冲突
