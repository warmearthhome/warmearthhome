# Phase 1: 紧急修复任务清单

## P0 - 移除所有中文内容

### 任务 1: 修复导航菜单中的中文
**问题**：网站导航中出现中文（"商店"、"我的帐户"、"结账"、"购物车"）

**原因分析**：
- WordPress/WooCommerce 语言包设置为中文
- 主题（Blocksy）语言设置为中文
- 需要将所有语言设置改为英文

**解决方案**：
1. 在 WordPress 后台：**设置 > 常规 > 站点语言** → 改为 `English (Australia)` 或 `English (United States)`
2. 在 WooCommerce：**WooCommerce > 设置 > 高级 > 页面** → 检查所有页面语言
3. 在 Blocksy 主题：**外观 > 自定义 > 语言** → 确保为英文
4. 清除所有缓存

### 任务 2: 移除首页测试内容
**问题**：首页显示测试内容（"CSS样式验证测试区域"）

**解决方案**：
1. 检查首页模板是否正确应用
2. 移除或替换所有测试内容
3. 使用真实内容或优雅的占位符

### 任务 3: 全站语言一致性检查
**检查项**：
- [ ] 所有导航菜单项
- [ ] 所有按钮文本
- [ ] 所有表单标签
- [ ] 所有提示信息
- [ ] Footer 内容
- [ ] 错误提示信息

## 执行步骤

### Step 1: WordPress 语言设置
需要你在 WordPress 后台手动操作：
1. 进入 **设置 > 常规**
2. 将 **站点语言** 改为 `English (Australia)`
3. 保存更改

### Step 2: WooCommerce 语言检查
1. 进入 **WooCommerce > 设置**
2. 检查所有页面的语言
3. 确保所有产品相关文本为英文

### Step 3: 清除缓存
- 清除 WordPress 缓存（如果有缓存插件）
- 清除浏览器缓存
- 刷新 CDN 缓存（如果有）

### Step 4: 验证结果
访问网站，检查：
- [ ] 导航菜单是否全部为英文
- [ ] 按钮文本是否全部为英文
- [ ] 所有页面内容是否无中文
